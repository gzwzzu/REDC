# REDC
Achieving sustainable development requires transforming development modes. This study examines the impact of China's Rural E-commerce Demonstration County (REDC) policy on common prosperity, using the Three Gorges Reservoir area as a case. We develop a theoretical framework and measurement index for common prosperity, and employ a multi-phase difference-in-differences model to evaluate the policy as a quasi-natural experiment. Results show the REDC policy significantly enhances common prosperity (robust at the 5% level), with mechanism tests revealing a 42.4% positive effect through income enhancement (significant at 1%). The policy operates via multiple channels, including output effects. This research provides empirical evidence on how regional e-commerce policies can foster common prosperity, offering insights for optimizing rural e-commerce in ecologically sensitive or underdeveloped regions and aligning industrial development with welfare distribution.

## 📁 Code and Implementation

* **[STATA DO code](./STATA%20DO%20code.zip)** This archive contains the core econometric scripts for the study. It includes the complete workflow for multi-phase difference-in-differences (DID) models, covering baseline regressions, robustness tests, and mechanism analysis. These files enable full replication of the empirical results regarding the impact of Rural E-commerce Demonstration County (REDC) policies on common prosperity.

* **[Geographical visualization code (Pyecharts)](./Geographical%20visualization%20code-Pyecharts)** This folder provides Python scripts utilizing the Pyecharts library for spatial data visualization. The code transforms raw data from the Three Gorges Reservoir area into geographic visualizations, illustrating the spatial evolution, distribution trends, and development patterns of various counties under the influence of regional e-commerce policies.
